## 特性

-   **纯前端实现**：所有工具均在浏览器端运行，无需后端服务器支持，响应速度极快。
-   **隐私安全**：数据不会上传到服务器，所有处理均在本地完成，保障数据安全。
-   **开箱即用**：无需注册登录，打开网页即可使用。
-   **现代界面**：采用深色科技风设计，支持响应式布局，适配桌面端与移动端。
-   **流畅体验**：精心设计的交互细节与动画效果，提升使用愉悦感。

## 本地运行

本项目为纯静态页面，你可以通过以下步骤在本地快速运行：

1.  **克隆仓库**
    ```bash
    git clone https://github.com/your-username/toolbox.git
    ```

2.  **进入目录**
    ```bash
    cd toolbox
    ```

3.  **打开文件**
    直接双击 `index.html` 在浏览器中打开，或者使用 VS Code 的 Live Server 插件启动。
